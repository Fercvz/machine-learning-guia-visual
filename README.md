# Machine Learning no Dia a Dia

Manual visual e interativo de Machine Learning voltado para projetos reais em supermercado, construção civil e varejo.

## Conteúdo

- fluxo prático do problema ao monitoramento;
- projetos completos de previsão de demanda e risco de atraso;
- laboratórios SVG interativos;
- biblioteca de teoria, incluindo aprendizado supervisionado e não supervisionado;
- trilha introdutória separada para SQL e construção de datasets;
- exemplos executáveis em Python com `scikit-learn`.

## Como usar

O projeto é um único arquivo HTML autocontido. Abra [`index.html`](./index.html) em um navegador moderno. Não há instalação, build ou servidor obrigatório.

## Estrutura das abas

- **ML no dia a dia:** decisões, validação, projetos e produção.
- **Teoria:** tipos de aprendizado, modelos, métricas e preparação dos dados.
- **SQL & Dataset:** visão geral da futura trilha de construção do dataset.

## Tecnologias

HTML, CSS, JavaScript e SVG, sem dependências externas de execução.
